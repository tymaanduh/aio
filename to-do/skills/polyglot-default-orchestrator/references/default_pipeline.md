# Default Pipeline

1. Intake and constraints capture
2. Stage planning (create vs maintain + stale stage detection)
3. Wrapper preflight (two-pass identify/execute validation)
4. English+pseudocode blueprint generation
5. Language feasibility and weighted scoring
6. Polyglot translation with shared behavioral contracts
7. Optimization/fix loop per language
8. Data-separation audit for object/name/group constants
9. Strict tests + security + benchmark gate
10. Recommendation and deliverable summary

## Standard Artifacts

- `plan/blueprint_english.md`
- `plan/blueprint_pseudocode.md`
- `analysis/wrapper_preflight_report.json`
- `analysis/data_separation_audit_report.json`
- `analysis/language_scorecard.json`
- `build/polyglot_manifest.json`
- `reports/sxs_benchmark_report.json`
- `reports/final_recommendation.md`
