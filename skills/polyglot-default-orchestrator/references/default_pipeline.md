# Default Pipeline

1. Intake and constraints capture
2. English+pseudocode blueprint generation
3. Language feasibility and weighted scoring
4. Polyglot translation with shared behavioral contracts
5. Optimization/fix loop per language
6. Strict tests + security + benchmark gate
7. Recommendation and deliverable summary

## Standard Artifacts

- `plan/blueprint_english.md`
- `plan/blueprint_pseudocode.md`
- `analysis/language_scorecard.json`
- `build/polyglot_manifest.json`
- `reports/sxs_benchmark_report.json`
- `reports/final_recommendation.md`
